# SmashLive Product

Product repository for the SmashLive project. This is where the PM agent stores specs, memory, and decisions.

## Structure

- `specs/` — Approved feature specs (written by PM agent on stakeholder approval)
- `memory/` — PM's accumulated project knowledge (domain, business rules, tech context)
- `decisions/` — Architecture Decision Records and business decisions

## PM Agent

The PM agent interacts with stakeholders via Linear and commits approved specs here.
