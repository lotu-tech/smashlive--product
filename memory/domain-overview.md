# SmashLive Domain Overview

SmashLive is a live sports streaming platform.

## Repos
- smashlive--app: Expo React Native mobile app
- smashlive--backoffice: React Vite admin dashboard
- smashlive--infra: Terraform + Node infrastructure
- smashlive--gcp--infra: Packer + Terraform GCP infrastructure
- smashlive--create-youtube-stream: TypeScript Docker Lambda
- smashlive--replay: TypeScript Docker Lambda
- smashlive--button: Arduino ESP32 physical button
- smashlive--youtube-credentials: React Vite YouTube credentials app
